高阶组件简单说就是一个抽象函数，用于复用组件逻辑。将组件的公共部分抽离出来封装成一个函数，该函数传入一个组件和对应的变量，新写的组件调用这个函数即可。新的组件=老的组件+新的state/props【以props传给老组件】。
